What is Stuff Plus Simple?


Stuff Plus Simple is a simple stuff plus model. It was built to help me understand how stuff models work. Stuff models try to quantify the value of a pitch solely on its physical attributes. Here is the link to the python project that generated all of the data on this website, and here is the link to the source code of this website.
